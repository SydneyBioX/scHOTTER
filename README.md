# scHOTTER

The goal of scHOTTER is to facilitate a parametric test for spatially heterogeneous co-expression in single-cell transcriptomics data, across trajectory, spatial and more contexts. This method and package were created by Liam Skender under the supervision of Dr. Shila Ghazanfar during 2025. 

## Installation

You can install the development version of scHOTTER using `devtools::install_github("https://github.com/SydneyBioX/scHOTTER")`.
